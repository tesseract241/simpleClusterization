This function calculates the weights of a fuzzy c-means clusterization, according to the next formula: w_ij = 1 / (norm(centroids(i), entities(j)))

float daviesBouldinIndex ( const Ref< const MatrixXf > &  entities,

Returns a measure of the fit of the clusterization, it's strictly positive and the smaller it is, the best the fit.

The Davies-Boulding Index defines a measure of the "goodness" of a clusterization of a data population based on the following quantities: The scatter vector S_i= (1/T_i * sum_j (norm(C_i, X_j)))^(1/2) where T_i is the population size of the i-th cluster and the sum runs over the datapoints belonging to the i-th cluster The Cluster Separation Matrix M_ij = (norm(C_i, C_j))^(1/2) The Davies-Bouldin Matrix R_ij = (S_i + S_j)/M_ij The Davies-Bouldin Vector R_i = max_(j!=i) R_ij The Davies-Bouldin index is, in terms of the previous quantities, R = 1/N * sum_i R_i where N is the number of clusters

Parameters
[in]entitiesThe datapoints
[in]centroidsThe centroids of the clusters
[out]weightsThe weights associating each centroid to its cluster (it's an array of bools)
[in]normA pointer to the norm function you want to use
Returns
The Davies-Boulding index of the provided clusterization
